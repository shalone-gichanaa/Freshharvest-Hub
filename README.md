# FreshHarvest Hub

FreshHarvest Hub is a simple web-based application built using **HTML, CSS, and JavaScript**.  
It connects local farmers with vendors and customers by showcasing fresh farm produce and allowing users to search and place simple orders.



## Purpose of the Project
The goal of this project is to support local farming by creating a simple digital space where:
- Farmers can list fresh produce
- Vendors can place bulk orders
- Customers can buy small household portions


## Features

### Search Functionality
Users can search for available produce such as spinach, sweet potatoes, matoke, and bananas. Results update instantly.

### Data Display
Each item shows:
- Produce name  
- Price per kilogram (KES)

### Forms & Interaction
- Search input filters results dynamically  
- Order forms allow users to simulate purchases  
- Input validation prevents empty submissions  

### DOM Manipulation
The page updates in real time without reloading:
- Search results update instantly  
- Order confirmations appear dynamically  

### Mock Data
The app uses a built-in JavaScript dataset to simulate real farm inventory.

### Styling & User Experience
The interface is designed to be simple, clean, and easy to use, with clear buttons and readable layouts.

## How to Run the Project

1. Download or clone this repository
2. Open the folder
3. Open `index.html` in any browser
4. Navigate between pages using the menu
5. Try searching or placing orders
